## Why? 

Backup and sync of my configs.

## How? 

I use [stow](https://www.gnu.org/software/stow) to manage my dotfiles.

```bash
# Install program
stow <directory>

# Remove program
stow -D <directory>
```

## Some of the programs I currently use:

- Wayland: `niri`
- Terminal: `foot`
- Shell: `tmux` and `zsh` with `powerlevel10k`
- Editor: `neovim` - based on `LazyVim`
- Bar: `eww`
- Launcher: `rofi`
